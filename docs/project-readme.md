## Что отправлять кому

### Инвестору (холодное письмо)
Шаблон A/B/C/D из `Email_шаблоны_инвесторам_v1.md` →
прикрепить `INERTUM-onepager.pdf` →
ссылка на `portal/public/INERTUM-pitch-v2.html` + `INERTUM-FAQ-investors.html`

### Инвестору после интереса (NDA + DD)
Шаблон F →
прикрепить `INERTUM-NDA.html` (распечатать в PDF) →
после подписания — выдать ключ через админку → инвестор открывает data room с патентом v2.1, тех.справкой, КД v2, финмоделью.

### Производителю (NomexRus, Pack-Systems)
Прикрепить `КД_Альбом_v1.html` (печать в PDF) + 4 SVG чертежа + RFQ-шаблон из `Поставщики_и_RFQ_v1.md`

### Патентному поверенному
Прикрепить `Патентная_заявка_драфт_v2_2026-05-08.md` + `Проверка_имени_INERTUM_2026-05-09.md` + просьба сделать ТМ-поиск в Роспатенте + WIPO

### Коллекционеру / клиенту (B2C продажа)
Ссылка на `portal/index.html` → секции Problem / Solution / Segments / Contact → форма запроса по email
