4. Если `.ru` свободен — взять обязательно (для Роспатента и российского B2G канала).

**Опция «выкупить .com»:**

- Решать, когда привлечём pre-seed раунд — до раунда не критично.

## 2. Поиск по бренду «INERTUM»

### `inertum.ae` — реальный бизнес в Дубае

WebFetch показал работающую страницу. Описание: **«coffee and cocoa trading»**, базируется в Дубае. По прямому WebSearch **`"INERTUM" Dubai coffee cocoa`** компания **не индексируется** в Google — нет упоминаний на yellowpages, DMCC, отраслевых каталогах. Возможные интерпретации:
- Очень маленький бизнес или ИП-форма, недавно открытый
- Неактивный landing-сайт (зарезервированное имя)
- Площадка под перепродажу

**Импликация для нас:**
- **По классам Nice они в 30 (кофе) / 35 (торговля).** Мы — в **17 / 20 / 22 (упаковка, тара, защитные материалы)**. По принципу различных классов ТМ — **бренды не конкурируют** и могут сосуществовать (пример: «Apple» в технике × «Apple» в музыке).
- Однако визуально и в общем поле бренда — путаница возможна. Особенно в MENA-канале, где наш `.ae` — занят.

### Прямой Google-поиск

Запросы:
- `"INERTUM" company brand product` → Google нашёл **Inert Products LLC** (США, военные тренажёры) — другое имя, не коллизия
- `"INERTUM"` отдельно — нет первой страницы крупных результатов
- `"INERTUM" trademark` — нет российских/международных регистраций в открытой выдаче

**Вывод:** имя малоизвестное, конфликтных активных брендов в open-web нет.

### WIPO Global Brand Database

Прямой поиск через `branddb.wipo.int/.../INERTUM` **под капчей**. В открытом гугле никаких международных Madrid-регистраций на `INERTUM` не индексируется.

**Что делать:** добавить в задачу патентному поверенному — поиск по WIPO Madrid + EUIPO + USPTO. Стоит ~₽10–20k, делается за неделю.

### EUIPO / USPTO

Без капчи поиск тоже невозможен с сервера. По косвенным признакам (Google indexing) — крупных регистраций нет.

## 4. Семантика и юридический аспект

**Этимология:** `inertum` — латинское прилагательное среднего рода от `iners` («инертный, неподвижный, бездействующий»). Совпадает с производственным смыслом — «инертный газ», «инертная среда». Семантически чистое для нашего поля.

**Описательность vs. фантазийность:** в патентной практике «INERTUM» считается **слабо-описательным** для класса 17 (промышленные материалы), **фантазийным** для класса 20/22 (упаковка). Это значит — регистрируется без проблем как ТМ для упаковки/тары/изделий.

**Класс Nice 17:** «Каучук, гуттаперча, резина, полимерные плёнки и тара…» — основная категория для нас.
**Класс Nice 20:** «Контейнеры неметаллические, ёмкости из пластмасс…»
**Класс Nice 22:** «Чехлы, обёрточные материалы, упаковочные мешки…»

**Рекомендация:** заявить ТМ в классах **17, 20, 22** (упаковка) + **9** (электронные сенсоры, опционально) + **35** (торговля упаковкой). В этих классах коллизий с дубайским кофе нет.
