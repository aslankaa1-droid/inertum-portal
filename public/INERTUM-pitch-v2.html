Богатые активы гниют, потому что упаковки для них не существует

Существующие решения работают на 5–10 лет, не на 50. Между ними — слепая зона на $50B+ премиум-активов.

Почему существующие решения не работают

  • CarCapsule, Carcoon — гонят воздух с фильтром, нужно электричество 24/7, внутри 21% O₂
  • Cortec, Zerust VCI-плёнки — летучие ингибиторы, ограничены 5–10 годами, не для электроники и фармы
  • MIL-STD-2073 / ГОСТ 9.014 — для отдельных агрегатов, не для целых изделий, без мониторинга герметичности
  • Анокс-витрины музеев — штучное производство $50k+, нет серийного продукта

§02 · SOLUTION 03 / 14

Гибкая капсула + инертный газ + абсорбер + пассивный оптический сенсор

5 слоёв защиты в одной капсуле. 50 лет хранения. Без батарей. Без радиосигнала. Без обслуживания.

LAYER 01

Барьерный ламинат

PET 12 / Al-foil 9 / LLDPE 70 мкм. OTR ≤ 0.05. MIL-PRF-131K, ГОСТ 9.014.

LAYER 02

Инертная среда

N₂ или Ar 99.99%. O₂ ≤ 0.5%. Точка росы ≤ −45 °C.

LAYER 03

Кислород-абсорбер

Mitsubishi Ageless Z. Дублирует газовый продув химически.

LAYER 04

Пассивный сенсор

PreSens SP-PSt6 ⌀5 мм. 5–7 лет автономно. Без батарей.

LAYER 05

Самодиагностика

Корреляция ΔP ↔ Δ[O₂] ↔ ΔT. Auto-alert при утечке.

PATENT

Patent Pending

Заявка ЕАПО v2.1: 3 indep + 20 dep claims. 21 проверенный аналог — блокеров нет.
