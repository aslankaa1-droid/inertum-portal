## Главный санкционный фильтр

Проект работает в РФ с возможностью госзаказа и dual-use. Это **жёстко режет западный/европейский pool**:

- **Sberbank, VTB Bank** — под полными OFAC / EU / UK блокирующими санкциями (CAPTA, SDN). Деньги через них принимать можно только рублёвым контуром в РФ.
- **Rostec** — под SDN США и санкциями ЕС (для рублёвого деала всё равно работоспособен).
- **Almaz Capital, Runa, Target Global, Fort Ross** — ушли из РФ-периметра (анкер EBRD запретил инвестировать в РФ).
- **Mubadala, PIF** — формально не санкционные, но compliance видит конечного бенефициара через структуру.

**Реалистичный фокус:** РФ-домашние институты, СНГ (Казахстан), MENA (UAE / KSA), Китай / HK, плюс частные русскоязычные ангелы вне санкционного пула.

**Структура:** SPV в Дубае или Алматы критична для приёма UAE / CN-капитала.

## Заметки по особенностям захода

1. **Warm intro обязателен:** HongShan, Mubadala, PIF, Granite Asia, Wa'ed (на топ-уровне). Без интро формы игнорируются.
2. **Сайтовые формы — реальный канал:** Phystech, Kama Flow, Astana Hub, Shorooq, Wa'ed, Wamda отвечают на качественные deck.
3. **Прямые email работают** у ФРП, Морейниса, Wamda, 500 MENA (через акселератор).
4. **Sber500** — это акселератор+ангел-чек, подавать через SberUnity, отдельный deeptech-трек без выручки.
5. **Dual-use disclosure:** для UAE/KSA/HK скрывать военный трек в pitch; продавать гражданскую упаковку / IoT / материалы. Военный — параллельно через ФРП / Минпромторг.
6. **Казахстан = безопасный санкционный буфер** для приёма $UAE / $CN.
