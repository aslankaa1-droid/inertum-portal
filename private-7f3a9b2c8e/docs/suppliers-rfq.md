## Часть 2. Беспроводные / оптические датчики O₂

### Главный архитектурный вывод эксперта

Стандартная схема «электрохимический O₂-сенсор + микроконтроллер + BLE + батарейка внутри капсулы» **не работает** для долгосрочного хранения по трём причинам:

| Проблема | Почему критична |
|---|---|
| **Деградация электрохимии** | Свинцово-анодные ячейки живут 2–3 года в воздухе; в чистом O₂ ещё быстрее. На 5–10 лет не подходят. |
| **Ресурс батареи** | CR2032 (220 мА·ч) хватит на 5 лет только при duty-cycle <0,01 %. ER14250 / ER18505 (Li-SOCl₂) — реалистично 7–10 лет, но габарит больше. |
| **Faraday cage** | Алюминиевая фольга оболочки гасит BLE 2,4 ГГц на 30–60 дБ. Связи через стенку **не будет**. |

**Решение** — двухуровневая оптическая архитектура:

```
┌──────────────────────── КАПСУЛА ────────────────────────┐
│                                                         │
│    [Оптический spot O₂ 5 мм] ← клеится изнутри на       │
│         (PreSens SP-PSt6-NAU)   маленькое прозрачное    │
│                                 окно (PET без фольги)   │
│                                                         │
│         НИКАКОЙ электроники, НИКАКОЙ батареи            │
│                  ↑                                      │
└──────────────────│──────────────────────────────────────┘
                   │ оптический сигнал
                   │ (читается через стекло/PET)
                   ↓
       [Внешний ридер: PreSens OXY-1 ST mini]
       подносится к окну при инвентаризации
       раз в неделю / месяц / квартал

   опционально:
       [Внешний автономный ридер на крышке капсулы]
       с LuminOx LOX-02-S + LoRa SX1262 + Li-SOCl₂
       Передаёт раз в сутки. Срок 7–10 лет.
```

## Часть 5. Ключевые риски и митигации

| Риск | Митигация |
|---|---|
| **NomexRus откажет или задержит** | Параллельно RFQ в Pack-Systems + Hangzhou Keneng (резерв) |
| **Логистика PreSens из ЕС задержится** | Параллельно PyroScience; резерв — собственная сборка LuminOx + Promwad |
| **Цена выше бюджета** | Уменьшить размер прототипа (взять не мотоцикл, а тестовый макет 2 × 0,8 × 1,3 м); сократить число спотов до 3 |
| **Faraday cage блокирует BLE даже у внешнего ридера** | PreSens-схема не использует радио вообще — проблема снимается архитектурно |
| **Электрохимия деградирует за 2 года** | Использовать только оптику (PreSens / PyroScience / LuminOx), не электрохимию |
| **Санкции на сенсоры** | Работаем через Турцию/Армению/Казахстан; в крайнем случае собственная сборка на российских компонентах + Wiren Board |
