## Слайд 2 — Problem

> **Богатые активы гниют, потому что упаковки для них не существует.**

| Сегмент | Что портится | Стоимость потерь |
|---|---|---|
| Коллекционные авто | Коррозия, окисление резины и кожи | $20B страхового рынка, ~5–10% потерь стоимости/год при сезонном хранении (Hagerty) |
| Музейные артефакты | Окисление металлов, выцветание | До 15% коллекций недоступно публике из-за деградации (ICOM) |
| Стратегические резервы РФ | Коррозия, потеря годности | Росрезерв и Минздрав теряют миллиарды ₽/год на пересортицу |
| Фарма / вакцины | Деградация активных веществ | До 15% годной продукции списывается (МВС Engineering) |
| Премиум-вино / виски / часы / ювелирка | Окисление, потеря товарного вида | Премиум-сегмент, маржа 70–90% |

**Существующие решения работают на 5–10 лет, не на 50.**
- CarCapsule, Carcoon — гонят воздух с фильтром, нужно электричество 24/7, 21% O₂ внутри.
- Cortec/Zerust VCI — летучие ингибиторы, 5–10 лет, не работают для электроники и фармы.
- MIL-STD-2073 — для агрегатов, не для целых изделий, без мониторинга.
- Анокс-витрины музеев — штучное производство $50k+, нет стандартизации.

## Слайд 3 — Solution

> **Гибкая герметичная капсула + инертный газ + кислород-абсорбер + оптический сенсор через окно. 50 лет хранения.**

Архитектура:

```
┌────────────────────────────────────────────────────┐
│ Многослойный ламинат PET / Al-foil / LLDPE         │
│ OTR ≤ 0.05 cc/m²/day · ГОСТ 9.014 ВУ-5             │
│                                                    │
│   [Инертный газ N₂ или Ar, O₂ ≤ 0.5%]              │
│   [6 саше Mitsubishi Ageless Z-2000]               │
│                                                    │
│   ┌─────────┐                                      │
│   │  Spot   │ ← пассивный оптический патч PreSens  │
│   │  PSt6   │   срок жизни 5–7 лет, без батареи    │
│   └─────────┘                                      │
│        ↑                                           │
│   [Прозрачное окно 30×30 мм без фольги]            │
│        ↑                                           │
└────────│───────────────────────────────────────────┘
         │ оптическое считывание снаружи
         ↓
   [Ридер OXY-1 ST mini] · ручной обход / стационарный
```

**Что делает уникальным:**
1. Полимерно-фольговая оболочка с локальным **прозрачным окном** для оптического считывания.
2. **Пассивный** оптический сенсор внутри (без батареи, без радио).
3. **Алгоритм самодиагностики** через корреляцию давления и O₂ — раннее обнаружение утечки.
4. **Порт безразрывной дозаправки** газом без вскрытия капсулы.

**Эта комбинация запатентована (заявка ЕАПО + PCT в подготовке).**

## Слайд 8 — Business Model

**Три потока выручки:**

1. **B2C прямые продажи** премиум-капсул (Auto, Moto, Vault, Archive) — высокая маржа, прямые продажи через сайт + VIP-каналы.
2. **B2B контракты** на стратегические резервы (Росрезерв, Минздрав, Минобороны, частные банки) — тендеры на партии 1 000–10 000 штук.
3. **Лицензирование патента** — после получения патента и одного-двух референс-контрактов: продажа лицензий зарубежным производителям упаковки (роялти 3–5%).

**Дополнительно:**
- Услуга «под ключ» — выезд бригады, упаковка изделия на месте, гарантия 50 лет (премиум-наценка +30%).
- Сменные кислород-абсорберы по подписке (recurring revenue).

## Слайд 10 — Unit Economics (на примере INERTUM Auto)

| Статья | $ |
|---|---|
| Ламинат PET/Al-foil/LLDPE 30 м² × $15 | 450 |
| Жёсткий каркас (профиль + фитинги) | 200 |
| Молния YKK Aquaseal 8 м × $20 | 160 |
| Газ: N₂ 2 баллона | 100 |
| Кислород-абсорбер: 6 саше Ageless Z-2000 | 30 |
| Оптические патчи PreSens × 3 | 120 |
| Упаковка, сборка, логистика | 300 |
| **COGS** | **1 360** |
| Цена продажи розница | 7 000 |
| **Gross profit на капсулу** | **5 640** |
| **Gross margin** | **80.6 %** |

**Точка безубыточности по модели Auto:**
- OpEx (зарплаты, аренда, маркетинг, R&D) ≈ $250 тыс. / квартал
- Break-even: ~45 капсул в квартал по средней цене $7k
- Целевой объём 2-го года: 200 капсул/квартал
